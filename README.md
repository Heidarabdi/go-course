# Go Course
This is a simple Go course repository to learn the basics of Go programming language.

## Course Content (so far what covered)
- [Hello World](helloworld/helloworld.go)
- [Variables](variables/variables.go)



## Prerequisites
- Basic understanding of programming concepts.
- A text editor or IDE of your choice.
- Go installed on your machine.

## How to run the code
1. Make sure you have Go installed on your machine. You can download it from [here](https://golang.org/dl/).

2. Clone this repository to your local machine using the command:
   ```
   git clone 

3. Navigate to the project directory:
   ```
   cd go-course

4. To run a specific Go file, navigate to its directory and use the command:
   ```
    go run filename.go
